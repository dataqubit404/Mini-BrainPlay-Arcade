
# 🧠 Mini-BrainPlay-Arcade

BrainPlay Arcade is a mini arcade-style game collection that brings together three classic puzzle and strategy games in one place. The goal of this project is to provide fun, simple, and engaging games that challenge memory, logic, and problem-solving skills.

Players can choose from different games through a central menu and enjoy quick gameplay sessions without installing large or complex software. BrainPlay Arcade focuses on clean design, simple controls, and smooth gameplay.

---

# 🎮 Games Included

## 1. Memory Match
Memory Match is a classic card puzzle game that tests the player's memory and concentration.

### How it Works
- A grid of cards is placed face down.
- Each card has a hidden symbol.
- Players flip two cards at a time.
- If the cards match, they stay visible.
- If they do not match, they flip back.

### Objective
Match all card pairs using the fewest number of moves and in the shortest time possible.

### Skills Tested
- Memory
- Focus
- Pattern recognition

---

## 2. Tic Tac Toe
Tic Tac Toe is a simple yet strategic two-player game played on a 3×3 grid.

### How it Works
- Players take turns placing their mark (X or O) on the grid.
- The first player to align three marks in a row wins.

### Winning Conditions
- Horizontal line
- Vertical line
- Diagonal line

### Game Modes
- Player vs Player
- Player vs Computer (optional future feature)

### Skills Tested
- Strategy
- Logical thinking
- Planning

---

## 3. Maze Escape
Maze Escape is a navigation puzzle game where players must find their way out of a maze.

### How it Works
- The player starts at the maze entrance.
- The maze contains walls and paths.
- The goal is to navigate through the maze and reach the exit.

### Objective
Find the shortest path to the exit while avoiding dead ends.

### Skills Tested
- Problem solving
- Spatial awareness
- Navigation

# ✨ Features

BrainPlay Arcade includes several features designed to enhance gameplay:

- Multiple mini games in one application
- Interactive game menu
- Clean and simple user interface
- Restart and replay options
- Score and move tracking

The project is designed to be beginner-friendly while still demonstrating key programming concepts used in game development.
